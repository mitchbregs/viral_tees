image-compare
=============

Compare multiple images reporting similarity. From
https://gist.github.com/attilaolah/1940208

Usage::

    $ ./compare_images
    usage: ./compare_images image-file-1.jpg image-file-2.jpg ...


Requirements
------------

Needs:
    * Pillow>=2.1.0 (PIL fork)
    * python-Levenshtein>=0.10.2
